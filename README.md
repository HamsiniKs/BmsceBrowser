# 🎓 Smart Campus Platform — BMSIT&M

A modern, interactive university website designed to transform how students explore campus life, academics, and opportunities.
Built with a focus on **data-driven insights, immersive UI/UX, and real-time interactivity**, this platform reimagines the traditional college website into a **premium digital experience**.

---

## 🚀 Overview

The Smart Campus Platform integrates multiple high-impact features into a single cohesive system:

* 📊 Placement Analytics Dashboard
* 🪑 Smart Seat Finder (Library & Canteen)
* 🎓 Department Explorer
* 🏆 Student Achievements Showcase
* 🎮 Interactive Event Tracker with Calendar
* 🎭 Club Explorer (Swipe-Based Discovery)
* 👩‍🎓 Alumni Journey Timeline

Each module is designed to be **interactive, visually engaging, and user-centric**, moving beyond static content.

---

## 🎯 Key Features

### 📊 Placement Analytics Dashboard

* Interactive India map with heatmap visualization
* City-level drilldowns with insights
* Filters by branch, year, and company type
* AI-generated placement insights
* Dynamic charts (bar, pie, line)

---

### 🪑 Smart Seat Finder

* Real-time seat availability (Library & Canteen)
* Movie-ticket style seat selection UI
* Heatmap view for crowd density
* Smart “Find Best Seat” recommendation
* Optional time-slot booking system

---

### 🎓 Department Explorer

* Tab-based navigation for all engineering branches
* About section with key stats and visuals
* Interactive faculty cards with hover details
* Structured and clean layout for easy exploration

---

### 🏆 Student Achievements Showcase

* **Spotlight Carousel** for featured achievements
* Filterable categories (Hackathons, Research, Internships)
* Interactive cards with detailed modal view
* Animated stats highlighting department success

---

### 🎮 Event Tracker + Calendar

* Interactive event discovery system
* Sidebar calendar for date-based filtering
* Category filters (Tech, Cultural, Sports)
* Dynamic event display with smooth transitions
* “Happening Now” and countdown indicators

---

### 🎭 Club Explorer (Swipe-Based)

* Tinder-style swipe interaction for discovering clubs
* Each card includes:

  * Club name
  * Description
  * Domain (Tech, Cultural, Sports, etc.)
* Swipe actions:

  * 👉 Right: Interested
  * 👈 Left: Skip
* Smooth card stacking animations
* Option to view detailed club info on tap

---

### 👩‍🎓 Alumni Journey Timeline

* Horizontal roadmap of alumni careers
* Interactive nodes (college → job → promotions)
* Search and filter functionality
* Career insights and comparison mode

---

## 🎨 Design Philosophy

* **Premium UI** inspired by Apple & Stripe dashboards
* **Dark mode first** with vibrant accent gradients
* **Glassmorphism + depth layering**
* **Collage & scrapbook aesthetics** for creativity
* **Micro-interactions** for a polished experience

---

## 🛠️ Tech Stack

* **Frontend:** React + Tailwind CSS
* **Animations:** Framer Motion
* **Charts:** Recharts / Chart.js
* **Maps:** SVG / react-simple-maps

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone <your-repo-url>
cd smart-campus
```

### 2. Install dependencies

```bash
npm install
npm install react@18 react-dom@18
```

### 3. Run development server

```bash
npm run dev
```

### 4. Open in browser

```
http://localhost:3000
```
